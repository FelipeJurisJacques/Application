Web APP C# Blazor

Limpar cache
dotnet clean

Compilar
dotnet build

Assistir
dotnet watch

Executar
dotnet run




* `Pages`: Pasta para armazenar os componentes Razor que representam páginas da aplicação. Cada página pode ter sua própria pasta ou ser uma única página Razor.
* `Components`: Pasta para armazenar os componentes reutilizáveis que podem ser utilizados em várias páginas.
* `Models`: Pasta para armazenar as classes de modelo de negócios que representam os dados da aplicação.
* `Services`: Pasta para armazenar as classes de serviço que lidam com a lógica de negócios e a comunicação com APIs, bancos de dados, etc.
* `Helpers`: Pasta para armazenar as classes de utilidade que fornecem funcionalidades auxiliares para a aplicação.
* `Shared`: Pasta para armazenar os componentes e recursos compartilhados entre várias páginas ou componentes.
